# Deskripsi Produk (Verifikasi Payment Gateway)

## Ringkasan Produk
KaraYouke adalah aplikasi karaoke berbasis web yang menghubungkan pemilik venue dan pengguna dalam satu platform. Aplikasi ini menyediakan pengalaman karaoke terintegrasi mulai dari pemesanan paket, pengelolaan ruangan, hingga kontrol pemutaran lagu secara real-time.

## Fitur Utama
- Manajemen paket karaoke dan harga.
- Pemesanan ruangan karaoke oleh pengguna.
- Dashboard admin untuk mengelola pengguna, ruangan, dan transaksi.
- Kontrol pemutar musik/playlist secara real-time untuk pengelola ruangan.
- Riwayat pembayaran dan status pembayaran di sisi pengguna.

## Alur Pembayaran
1. Pengguna memilih paket karaoke.
2. Pengguna melakukan pembayaran melalui payment gateway.
3. Sistem menerima status pembayaran (berhasil/gagal).
4. Jika berhasil, paket/akses ruangan diaktifkan dan ditampilkan pada dashboard.
5. Riwayat transaksi tersimpan dan dapat diakses pengguna/admin.

## Jenis Produk/Jasa
Produk digital berupa layanan pemesanan paket karaoke dan akses ruangan karaoke.
